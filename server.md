# Home Server with Coolify

Simple home server stack.
Click → deploy → use.

---

## Platform

* **[Coolify](https://coolify.io)**
  Self-hosted app platform

---

## Apps

* **[n8n](https://n8n.io)**
  Automation workflows

* **[Nextcloud](https://nextcloud.com)**
  Private cloud storage

* **[Supabase](https://supabase.com)**
  Open-source backend

* **[Uptime Kuma](https://github.com/louislam/uptime-kuma)**
  Service monitoring

* **[Jellyfin](https://jellyfin.org)**
  Media server

* **[LinkStack](https://linkstack.org)**
  Self-hosted link page

---

## What You Need

* Linux server
* Docker
* Storage for data
* Backup location
* Optional domain
