# Python Libraries (Simple Stack)

Clean, beginner-friendly tools for building Python apps fast.

---

## Core

* **pydantic**  
  Validate and structure your data 
  
* **typer**  
  Build CLI apps quickly

* **rich**  
  Better terminal output (logs, progress bars)

* **requests**  
  Simple HTTP requests

---

## Data & Files

  Work with tables and data easily

* **pillow**  
  Basic image processing

---

## Automation

* **tenacity**  
  Retry failed API calls automatically

---

## AI

* **openai**
