# After Install – Linux

Quick setup after installing Linux. Click through GUI.

---

## 1️⃣ Updates

* Open **Software → Updates → Install all → Restart**

## 2️⃣ NVIDIA Drivers

* **Menu → Software & Updates → Additional Drivers → Select NVIDIA → Apply → Restart**

## 3️⃣ Flatpak

* Open **Software**
* Enable Flatpak when prompted

## 4️⃣ Change DNS

* **Settings → Network → Active connection → IPv4 → DNS: 1.1.1.1 / 1.0.0.1 → Save**

## 5️⃣ Backups

* **Backups → Select files/location → Enable auto-backup → Run first backup**

---

✅ **Done**
System ready for daily use.
